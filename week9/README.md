# Задачи
## Задача 1
Напишете структурата едносвързан списък от цели числа и напишете за него следните функции
- `printList :: List -> String`
- `addElement :: List -> Int -> List`
- `appendList :: List -> List -> List`

Бонус: напишете функцията за принтиране така че да извежда елемените по следния начин
`1->2->3`, като не ползвате условен оператора, guards или case.
## Задача 2
Добавете следните функции и променливи към имплементацията на едносвързания списък
- `mapList :: (Int -> Int) -> List -> List` - имплементирайте `map` но върху нашите списъци
- `genList :: List` - генерира списък с всички естествени числа
- `take :: Int -> List -> List` - взима първите n елемента от списък
## Задача 3
Напишете структурата бинарно дърво от числа и имплементирайте следните функции.
- `printTree:: Tree -> String` - тази функция принтира дървото в следния вид `(( T1 ) a ( T2 ))` за дървото с корен a и поддървета T1 и Т2.
- `search :: Tree -> Int -> Bool` - връща дали дадена стойност се намира в дървото в логаритмично време (за балансирани дървета).
- `addElementBinaryTree :: Tree -> Int -> Tree` - добавя втория аргумент в бинарното дърво
## Задача 4
Напишете следните функции и променливи за дърво от символи
- `infiniteLanguageTree :: Tree` - създава безкрайно дърво където за елемент $\alpha$ 
    - лявата част на дървото съдържа всички стрингове от вида $\beta .a. \alpha$ за някое произволно $\beta$
    - дясната част на дървото съдържа всички стрингове от вида $\beta .b. \alpha$ за някое произволно $\beta$
## Задача 5
Напишете следните функции за дървото от горната задача
- `limitHeight :: Int -> Tree -> Tree` - лимитира височината на подаденото дърво до първия аргумент
- `mapTree :: (String -> String) -> Tree -> Tree` - прилага функцията върху всеки елемент на дървото
## Задача 6
Напишете следните функции и самата структура за полиморфен свързан списък.
- `printList :: Show a => List a -> String` - принтиране на списък
- `mapList :: (a -> b) -> List a -> List b` - преобразуване на списък
- `grapeList :: List a -> List String` - превръща всички елементи в списъка на думата grape
## Задача 7
Напишете нов дип за дроби, като имплементирате
- Събиране и изваждане
- Деление и умножение
- Преобразуване в стринг и равенство
## Задача 8
Напишете алгебричната структура за потребител с име и възраст и напишете следните функции
- `usersEqual :: User -> User -> Bool` - два потребители са с равни стойности
- `createUser :: String -> Int -> User` - създава потребител по име и възраст
- `getUserName :: User -> String` - взима името на потребителя
- `getUserAge :: User -> Int` - взима възраста на потребителя
- `setUserName :: String -> User -> User` - модифицира името на потребителя
- `setUserAge :: Int -> User -> User` - модифицира възраста на потребителя
## Задача 9
Добавете към горната задача възможноста да се пазят потребителите в асоциативен списък с ключ име и напишете следни функции
- `createPlatform :: Platform` - създава платформа
- `addUser :: Platform -> User -> Platform` - добавя потребител
- `getUser :: Platform -> String -> User` - взима потребител по име
- `removeUser :: Platform -> String -> Platform` - маха потребител
- `updateUser :: Platform -> User -> Platform` - ъпдейтва потребител от платформата
## Задача 10
На горната задача променете функцията getUser да връща `Maybe User`
- `getUser :: Platform -> String -> Maybe User` - взима потребител по име

Бонус:
- Променете да връща `Either String User`
